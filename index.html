Teste Visual: WOFF2 vs WOFF

Acima você verá dois blocos: “Somente WOFF2” (à esquerda) e “Somente WOFF” (à direita). Se ambos exibirem o ícone corretamente, significa que tanto o WOFF2 quanto o WOFF estão válidos. Se o bloco “WOFF2” aparecer quebrado (quadrado vazio), mas o “WOFF” aparecer OK, então o WOFF2 não está sendo carregado corretamente. Use o DevTools ↦ Network para ver qual arquivo foi baixado em cada caso.

Como interpretar:
• Se o navegador suportar WOFF2, o bloco “Somente WOFF2” deve exibir o ícone corretamente. • O bloco “Somente WOFF” também deve exibir o mesmo ícone (POSIÇÃO IGUAL) — isso mostra que o arquivo WOFF é válido. • Caso algum dos blocos mostre apenas um quadrado vazio (tofu), significa que ele não conseguiu carregar a fonte no formato correspondente.
